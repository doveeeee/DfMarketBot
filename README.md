# DeltaForceMarketBot

用于腾讯三角洲行动游戏内交易行的市场工具

# 部署

下载代码，安装requirements.txt

# 运行

```python
python DFMarketBot.py
```

等待命令行中显示''初始化完成''

启动循环前先输入理想价格和最高价格↓

![1745483382173](image/README/1745483382173.png)

**把鼠标放在要买的物品上**

F8启动循环开始自动购买，F9停止循环

# 购买逻辑

## 正常模式

如果底价低于理想价格就买200发

如果底价高于理想价格就买1发来刷新价格

如果底价高于最高价格就重进商品页面刷新价格

## 钥匙卡模式（测试阶段）

如果底价低于理想价格就买1张，然后结束循环

如果底价高于理想价格就重进商品页面刷新价格

# 画饼
